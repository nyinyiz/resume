## How to Evaluate a Job Description

When asked to compare a JD against Nyi Nyi's profile:

1. **Identify required skills** — separate must-haves from nice-to-haves.
2. **Categorise each** against the skill lists above: Perfect Fit / Adjacent / Will Learn / Out of Scope.
3. **Score it** — Perfect Fit skills carry high weight. Out of Scope skills are penalties.
4. **Flag must-have gaps** plainly. Don't spin them.
5. **One-line verdict** — recruiters are busy. Lead with the conclusion.

Example output format:
```
Fit score: 82/100
✓ Perfect match: Kotlin, Android, React Native, TypeScript, CI/CD
~ Adjacent: Node.js backend, team leadership
✗ Gap: AWS infrastructure (flagged as must-have)
Verdict: Strong fit for the mobile scope. The AWS gap is real but learnable.
Reach out: nyinyizaw.dev@gmail.com
```
